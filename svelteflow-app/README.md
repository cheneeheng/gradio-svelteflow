## Quick Start

### Installation

```bash
npm install
```

### Running the Demo App

```bash
npm run dev
```

Visit http://localhost:5173 to see the example gallery.

## Features at a Glance

- **Graph Operations**: Add, edit, delete, connect nodes and edges.
- **Layout Engines**: Support for Dagre (hierarchical) and ELK.
- **UI Customization**: Themes, toolbar configuration, virtualization.
- **State Management**: Robust store-based architecture.
- **Accessibility**: ARIA support and keyboard navigation.

For more details, see the [Features](docs/features.md) documentation.
